# REST API Server Mock

Макет сервера REST API - pre-alpha

## Запус и управление

- собрать контейнер `make build`
- стартовать приложение `make start`
- остановить приложение `make stop`
- сервер запущен (по умолчанию слушает на `8088` порту)
- reload приложения `make reload`
- посмотреть логи приложения `make logs`
- инициализировать БД:
    - убедиться, что приложение запущено (docker-compose ps), если нет - стартовать приложение
    - выполнить команду инициализации БД `make db-init`

## API

- `GET http://localhot:8088/api/1.0/projects/<group_id>/` - список проектов группы (group_id = 0 - все проекты)
- `GET http://localhot:8088/api/1.0/groups/` - список групп (фиктивная урла, для проверки закрытых от паблика api)
- `POST http://localhot:8088/api/1.0/auth/login/` - авторизоваться на серевре (ожидаются параметры login, password) в ответ приходит token
- `POST http://localhot:8088/api/1.0/auth/logout/` - завершить авторизованную сессию работы
